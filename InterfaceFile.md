{

## 本地钱包
### 创建钱包
请求:
 
 参数 | 释义 | 默认值 | 是否必传 | 备注
 ---- | ----- | ------ | ------  | ------  
 wPwd  | 钱包密码 | 无 | 是 | 无   
 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：
  
 字段 | 释义 | 备注
 ---- | ----- | ------ 
 eAddress  | eth地址 | 无
 publicKey  | 公钥 | 加密后返回
 privateKey  | 私钥 | 加密后返回 
 keyStore  | keyStore | 加密后返回 
 
 ### 导入钱包
请求:
 
 参数 | 释义 | 默认值 | 是否必传 | 备注
 ---- | ----- | ------ | ------  | ------  
 wWord  | 助记词 | 无 | 是 | 无 
 wPwd  | 钱包密码 | 无 | 是 | 无   
 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：
  
 字段 | 释义 | 备注
 ---- | ----- | ------ 
 eAddress  | eth地址 | 无
 publicKey  | 公钥 | 加密后返回
 privateKey  | 私钥 | 加密后返回 
 keyStore  | keyStore | 加密后返回 
 
### 获取余额
请求: 数组形式
 
 参数 | 释义 | 默认值 | 是否必传 | 备注
 ---- | ----- | ------ | ------  | ------  
 eAddress  | eth地址 | 无 | 是 | 无 
 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：数组形式
  
 字段 | 释义 | 备注
 ---- | ----- | ------ 
 currency  | 币种 | 无
 balance  | 余额 | 无 
  
### 转账
请求:
 
 参数 | 释义 | 默认值 | 是否必传 | 备注
 ---- | ----- | ------ | ------  | ------  
 type  | 转账类型 | 无 | 是 | 0 其他钱包 1 云钱包
 fromAddress  | 本地钱包地址 | 无 | 是 | 无
 privateKey  | 私钥 | 无 | 是 | 加密
 toAddress  | 转账地址 | 无 | 否 | 其他钱包传
 uId  | 用户Id | 无 | 否 | 云钱包传
 limit  | 矿工费 | 无 | 是 | 无
 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：
   
 字段 | 释义 | 备注
 ---- | ----- | ------ 
 result  | 转账发起是否成功 | 无

## 我的
### 发送验证码
请求:
 
 参数 | 释义 | 默认值 | 是否必传 | 备注
 ---- | ----- | ------ | ------  | ------  
 phone  | 手机号 | 无 | 是 | 无 
 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：
  
 字段 | 释义 | 备注
 ---- | ----- | ------ 
 result  | 是否成功 | 无
 
### 登录/注册
请求:
 
 参数 | 释义 | 默认值 | 是否必传 | 备注
 ---- | ----- | ------ | ------  | ------  
 phone  | 手机号 | 无 | 是 | 无 
 verificationCode  | 验证码 | 无 | 是 | 无 
 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：
  
 字段 | 释义 | 备注
 ---- | ----- | ------ 
 uId  | 用户Id | 无 
 uName  | 用户名 | 生成一个不重复随机名

 sign  | 参数加密 | 无 | 是 | 无   
 
 返回：

 ---- | ----- | ------

 uName  | 用户名 | 无 
 ubalance  | 用户余额 | 数组
 
  字段 | 释义 | 备注
 ---- | ----- | ------
